🖥️ CLI 指令與旗標
核心指令
claude 互動模式
claude "q" 帶提示
claude -p "q" 無頭模式
claude -c 繼續上次
claude -r "n" 恢復
claude update 更新
重要旗標
--model 設定模型
-w Git Worktree
-n / --name 工作階段名稱
--add-dir 新增目錄
--agent 使用 Agent
--allowedTools 預先核准
--output-format json/stream
--json-schema 結構化
--max-turns 限制輪次
--max-budget-usd 費用上限
--console 透過 Anthropic Console 驗證
--verbose 詳細模式
--bare 最小無頭模式（無 hooks/LSP）NEW
--channels 權限轉發 / MCP 推送NEW
--remote 網頁工作階段
--chrome Chrome
權限模式： default 提示確認 · acceptEdits 自動接受編輯 · plan 唯讀 · dontAsk 拒絕（除非已允許） · bypassPermissions 略過所有
重要環境變數： ANTHROPIC_API_KEY · ANTHROPIC_MODEL · CLAUDE_CODE_EFFORT_LEVEL（low/med/high） · MAX_THINKING_TOKENS（0=關閉） · CLAUDE_CODE_MAX_OUTPUT_TOKENS（預設 32K） · CLAUDE_CODE_DISABLE_CRON
